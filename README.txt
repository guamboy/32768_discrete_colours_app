Coding Challenge
32768 Discrete Colours app
Generate 32,768 unique colours. Create colours by combining a red, green, and blue component in 
the range of 0 to 256. The colours should change by units of 8. Colours are displayed in pixel 
format using SVG and Canvas.

App Type
Front end only using React.

Clone and install dependencies
git clone git@github.com:guamboy/32768_discrete_colours_app.git
cd 32768_discrete_colours_app
cd colour app 
yarn install

Run
yarn watch

Test
yarn test
